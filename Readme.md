# Resume Builder

A full-stack web application for creating and managing professional resumes.

## Tech Stack

- **Frontend:** Angular
- **Backend:** Java Spring Boot
- **Database:** PostgreSQL

## Architecture

```text
Angular (Vercel)
       │
       ▼
Spring Boot API (Render)
       │
       ▼
PostgreSQL (Supabase)
```

## Features

- Create resumes
- Edit resumes
- Save resume data
- Responsive UI
- RESTful APIs

## Run Locally

### Frontend

```bash
cd frontend
npm install
ng serve
```

### Backend

```bash
cd backend
mvn spring-boot:run
```
